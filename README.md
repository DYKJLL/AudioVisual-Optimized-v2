## 🚀 使用方法

### 一键启动版
1. 下载 `AudioVisual-1.2.4-x64.exe`
2. **双击直接运行** ✅
3. 无需安装，无需解压

### 特点
- ✅ 单个 exe 文件
- ✅ 所有资源打包在内
- ✅ 双击即可使用
- ✅ 便携版，可放 U 盘随身携带

## ✨ 功能特性

- 🎬 多平台视频解析
- 🌍 剧迷模式（美韩剧/日剧）
- ⚡ 自动解析播放
- 🎨 侧边栏快速导航
- ⚙️ 支持自定义解析接口
- 🔧 主题切换
