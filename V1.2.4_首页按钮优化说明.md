# V1.2.4 优化说明

## 修复日期
2026 年 4 月 15 日

## 修复内容

### 首页按钮同步逻辑优化

**问题描述：**
- 剧迷模式下点击首页按钮，页面上拉框不同步
- 地址栏 URL 可能不匹配实际页面
- 没有加载中提示，用户体验差
- 空站点时无任何提示

**修复内容：**
1. 剧迷模式点击首页 → 回到当前选中的站点（不再是固定第一个）
2. 同步下拉框到当前首页站点
3. 同步地址栏 URL
4. 添加"加载中..."动画提示
5. 空站点时弹窗提示用户去配置

**代码位置：**
- `assets/js/renderer.js` line 394-421

**修复前代码：**
```javascript
homeButton.addEventListener('click', () => {
    isCurrentlyParsing = false;
    const isDramaMode = container.classList.contains('drama-mode');
    
    if (isDramaMode) {
        const homeUrl = dramaSites.length > 0 ? dramaSites[0].value : '';
        if (homeUrl) {
            window.voidAPI.resetModule(homeUrl);
        }
    } else {
        // ... 解析模式逻辑
    }
});
```

**修复后代码：**
```javascript
homeButton.addEventListener('click', () => {
    isCurrentlyParsing = false;
    const isDramaMode = container.classList.contains('drama-mode');
    
    if (isDramaMode) {
        if (dramaSites.length === 0) {
            showToast('剧迷模式未配置站点，请先在设置中添加', 'warning');
            return;
        }
        
        const homeUrl = quickDramaSelect.value || dramaSites[0].value;
        
        quickDramaSelect.value = homeUrl;
        urlInput.value = homeUrl;
        currentVideoUrl = homeUrl;
        loadingOverlay.classList.remove('hidden');
        
        window.voidAPI.resetModule(homeUrl);
    } else {
        const homeUrl = platformSelect.value;
        if (homeUrl === 'https://www.youku.com') {
            youkuCustomPage.style.display = 'flex';
            window.voidAPI.setViewVisibility(false);
            urlInput.value = '';
        } else {
            loadingOverlay.classList.remove('hidden');
            window.voidAPI.resetModule(homeUrl);
        }
    }
});
```

**效果对比：**

| 修复前 | 修复后 |
|--------|--------|
| 下拉框不跟随页面变化 | 下拉框实时同步 |
| 地址栏可能不匹配 | 地址栏始终准确 |
| 无加载提示 | 显示加载中动画 |
| 空站点无提示 | 友好提示用户配置 |

**用户场景示例：**
```
场景：用户在剧迷模式下选择了"看片狂人"，然后点击首页按钮
修复前:
  ✅ 页面跳转到看片狂人
  ❌ 下拉框仍显示之前的选择（看片狂人/奈飞工厂混乱）
  
修复后:
  ✅ 页面跳转到看片狂人
  ✅ 下拉框同步显示"看片狂人"
  ✅ 地址栏更新为看片狂人 URL
  ✅ 显示"加载中..."动画
```

## 打包方式

开发完成后，在本地打包：

```bash
# 安装依赖
npm install

# Windows 打包
npm run dist:win

# macOS 打包
npm run dist:mac

# Linux 打包
npm run dist:linux

# 全平台打包
npm run dist:all
```

打包完成后，可执行文件在 `dist/` 目录下。
